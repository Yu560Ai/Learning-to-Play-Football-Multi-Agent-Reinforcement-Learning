# Configs

Put shared experiment configurations for `Two_V_Two` here.

Recommended use:

- keep only configs that are meant to be reused by the team
- do not store checkpoints, logs, or replay outputs here
- name files by purpose, for example:
  - `baseline_ppo.json`
  - `ppo_kl_prior.json`
  - `ppo_reward_shaping.json`
